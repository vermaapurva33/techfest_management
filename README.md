# Techfest Management System 

A comprehensive web-based system to manage technical college fests, including events, participants, sponsors, and payments.

## Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP (Apache)

## Setup Instructions
1.  Download and install XAMPP.
2.  Clone this repository into the `htdocs` folder:
    `C:\xampp\htdocs\`
3.  Start **Apache** and **MySQL** in the XAMPP Control Panel.
4.  Import the database file (`database/schema.sql`) using phpMyAdmin.
5.  Open browser and go to: `http://localhost/techfest_system/`
